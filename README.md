# handwritten-digit-spot-detection
A Python-based handwritten digit recognition application using a CNN model, featuring a Tkinter GUI that allows users to draw digits and receive real-time predictions with confidence after automated image preprocessing.
# Handwritten Digit Spot Detection 

A GUI-based handwritten digit recognition system using a CNN trained on MNIST.
Users draw digits on a canvas and receive real-time predictions with confidence.

## Features
- Interactive Tkinter GUI
- Automatic digit cropping and centering
- CNN-based digit classification
- Confidence score display

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Pillow
- Tkinter
