Model Explanation

This project uses a Convolutional Neural Network (CNN) trained on the MNIST dataset to classify handwritten digits.

Model Objective

Recognize handwritten digits from 0 to 9

Provide high-accuracy predictions on grayscale images

Model Architecture

Convolutional layers with ReLU activation

MaxPooling layers for spatial feature reduction

Fully connected (Dense) layers

Softmax output layer with 10 classes

Input Details

Image size: 28 × 28

Color mode: Grayscale

Input shape: (28, 28, 1)

Output Details

Digit class: 0–9

Probability distribution across all classes
